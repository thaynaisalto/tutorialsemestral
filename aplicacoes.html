    • Na matéria de Aplicações para Internet aprendi que:


    Aplicações para Internet na faculdade abordam o desenvolvimento de sistemas e tecnologias para a web. Os alunos aprendem a criar sites, aplicativos e serviços que funcionam na Internet. Eles exploram linguagens de programação web, como HTML, CSS e JavaScript, além de frameworks e bibliotecas populares. O curso visa capacitar os alunos a projetar e implementar soluções web inovadoras e eficientes.


    Aprendizados do semestre:.


    • HTML (Tags semânticas, criação de pequenos sites e formulários);


    • Básico de CSS;


    • PHP básico;


    • Conectar o site em um servidor local (USBWebserver);


    • Criação de site e como hospedar no GITHUB (aprendizado pessoal para complementação da matéria).
