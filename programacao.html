    • Na matéria de Programação de Computadores aprendi que:


    A programação de computadores na faculdade envolve o estudo e a prática de linguagens de programação para criar e desenvolver software.


    Logo, aprendemos os fundamentos da lógica de programação, estruturas de dados e algoritmos para resolver problemas computacionais.


    A linguagem do semestre foi Python, nisso apredemos diversas coisas como:


    • O básico da estrutura da linguagem, como realizar cálculos matemáticos ou fazer inputs para comparativo;


    • Laço de repetição (FOR e While);


    • Funções (def);


    • Estrutura Condicional (IF, ELIF e ELSE);


    • Dentre outras.


    Realizamos o projeto indisciplinar, o objetivo era criar uma calculadora com Python onde o usuário escolhesse uma base (no nosso caso foi a decimal) e transcrever o número da base primária para: Octal, Binário e Hexadecimal.
