    Na reta final do primeiro semestre de Ciência da Computação, foi possível perceber a importância dos fundamentos essenciais para a área. Durante esse período, foram abordados conceitos de programação, lógica de programação, estruturas de dados e arquitetura de computadores. Essa base é fundamental para o desenvolvimento de habilidades técnicas e para a compreensão dos princípios que regem a computação. Com isso, estou mais preparada para o próximo semestre.


    Cada matéria teve seu ensinamento essencial para criar a base inicial, tanto as teóricas ensinando como é a estrutura de um computador ou como pensar igual uma máquina ao solucionar um problema, quanto matérias práticas ensinando como é montar um código do zero e programar de forma limpa. Particularmente adorei todas as matérias, nunca vou esquecer do meu primeiro "Hello World" no HTML e no Python.


    O site atual foi criado totalmente em base de todos os conhecimentos adquiridos em todas as matérias, sendo a mais importante Aplicações para Internet, utilizando como fonte de conhecimento o curso de HTML 5 e CSS3 na plataforma YouTube, no canal "Curso em Video" do Professor Gustavo Guanabara.
